Upload Customers

To upload customers to SlidePiper, follow these instructions:

1. Create a CSV file of your customers by one of these options:

Export a CSV file from Outlook: (How?)
-OR-
Export a CSV file from Gmail (Outlook CSV Format): (How?)

2. Click on the 'Choose File / Browse' button and select your CSV file.
3. Click on the 'Upload Customers' button to upload the CSV file to the system.
